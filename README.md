## ğŸš€ Quick Start

### Prerequisites

- **Node.js** 18.0.0 or higher
- **PostgreSQL** 14.0 or higher (or Neon account)
- **Clerk** account for authentication
- **Google AI** API key for Gemini

### 1. Clone Repository

```bash
git clone https://github.com/vipinyadav01/aicareercoach.git
cd aicareercoach
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Environment Setup

Create `.env.local` file in the root directory:

```env
# Database
DATABASE_URL="postgresql://username:password@host:port/database"
DIRECT_URL="postgresql://username:password@host:port/database"

# Clerk Authentication
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=pk_test_xxx
CLERK_SECRET_KEY=sk_test_xxx
NEXT_PUBLIC_CLERK_SIGN_IN_URL=/sign-in
NEXT_PUBLIC_CLERK_SIGN_UP_URL=/sign-up

# Google AI
GEMINI_API_KEY=your_gemini_api_key

# Inngest (Optional)
INNGEST_EVENT_KEY=your_inngest_event_key
INNGEST_SIGNING_KEY=your_inngest_signing_key
```

### 4. Database Setup

```bash
# Generate Prisma client
npx prisma generate

# Push database schema
npx prisma db push

# (Optional) Seed database
npx prisma db seed
```

### 5. Run Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## âš¡ Background Jobs with Inngest

LaunchTrack uses [Inngest](https://www.inngest.com/) for reliable background job processing:

### ğŸ”„ **Scheduled Jobs**

#### ğŸ“Š Weekly Industry Insights (`generateIndustryInsights`)
- **Schedule:** Every Sunday at midnight
- **Purpose:** Updates market data, salary ranges, and growth trends
- **AI-Powered:** Uses Google Gemini to analyze latest industry data

#### ğŸ’¡ Career Tips (`sendCareerTips`)  
- **Schedule:** Every Monday at 9 AM
- **Purpose:** Sends personalized career advice to active users
- **Personalized:** Based on user's industry, experience, and skills

#### ğŸ“ˆ Performance Analysis (`analyzeCoverLetterPerformance`)
- **Schedule:** Every Tuesday at 10 AM  
- **Purpose:** Analyzes cover letter creation trends and user engagement

### ğŸ¯ **Event-Driven Jobs**

#### ğŸ‘‹ New User Onboarding (`processNewUserOnboarding`)
- **Trigger:** `user/created` event
- **Delay:** 1 hour after user creation
- **Purpose:** Sends personalized welcome content and feature recommendations

#### ğŸ­ Interview Questions (`generateInterviewQuestions`)
- **Trigger:** `interview/practice-requested` event
- **Purpose:** Creates role-specific interview questions with AI
- **Custom:** Tailored to job title, industry, and experience level

### ğŸ›  **Usage Examples**

```javascript
import { triggerNewUserOnboarding, requestInterviewPractice } from '@/lib/inngest/events';

// Trigger when user completes registration
await triggerNewUserOnboarding({
  id: user.id,
  email: user.email,
  name: user.name
});

// Request interview practice questions
await requestInterviewPractice({
  userId: user.id,
  jobTitle: "Senior Software Engineer",
  industry: "Technology",
  experience: 5
});
```

### ğŸ”§ **Configuration**

## ğŸ“Š Database Schema

```prisma
model User {
  id              String           @id @default(uuid())
  clerkUserId     String           @unique
  email           String           @unique
  name            String?
  industry        String?
  experience      Int?
  skills          String[]
  assessments     Assessment[]
  coverLetter     CoverLetter[]
  resume          Resume?
}

model CoverLetter {
  id             String   @id @default(cuid())
  userId         String
  content        String
  jobDescription String?
  companyName    String
  jobTitle       String
  status         String   @default("draft")
  user           User     @relation(fields: [userId], references: [id])
}

model Assessment {
  id             String   @id @default(cuid())
  userId         String
  quizScore      Float
  questions      Json[]
  category       String
  improvementTip String?
  user           User     @relation(fields: [userId], references: [id])
}
```
